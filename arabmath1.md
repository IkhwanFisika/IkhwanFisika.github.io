### MSA Arabic Classroom Phrases Elementary Level
| **Speaker**  | **Arabic (with ḥarakāt)**                                                                                | **English Translation**                                    |
| ------------ | -------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| **Teacher**  | مَنْ يُمْكِنُهُ أَنْ يَحْسِبَ وَاحِدًا زَائِدَ وَاحِدٍ؟                                                  | Who can calculate one plus one?                            |
| **Student**  | وَاحِدٌ زَائِدٌ وَاحِدٌ يُسَاوِي اِثْنَانِ.                                                              | One plus one equals two.                                   |
| **Teacher**  | أَحْسَنْتَ!                                                                                              | Well done!                                                 |
| **Teacher**  | كَمْ يُسَاوِي ثَلَاثَةٌ نَاقِصٌ وَاحِدٌ؟                                                                 | What is three minus one?                                   |
| **Student**  | يُسَاوِي اِثْنَيْنِ.                                                                                     | It equals two.                                             |
| **Teacher**  | جَيِّدٌ جِدًّا!                                                                                          | Very good!                                                 |
| **Teacher**  | مَنْ يَعْرِفُ كَيْفَ نَجْمَعُ خَمْسَةً وَأَرْبَعَةً؟                                                     | Who knows how to add five and four?                        |
| **Student**  | خَمْسَةٌ وَأَرْبَعَةٌ يُسَاوِي تِسْعَةً.                                                                 | Five and four equals nine.                                 |
| **Teacher**  | رَائِعٌ!                                                                                                 | Excellent!                                                 |
| **Teacher**  | هَيَّا نَعُدَّ مَعًا مِنْ وَاحِدٍ إِلَى عَشَرَةٍ.                                                        | Let’s count together from one to ten.                      |
| **Students** | وَاحِدٌ، اِثْنَانِ، ثَلَاثَةٌ، أَرْبَعَةٌ، خَمْسَةٌ، سِتَّةٌ، سَبْعَةٌ، ثَمَانِيَةٌ، تِسْعَةٌ، عَشَرَةٌ. | One, two, three, four, five, six, seven, eight, nine, ten. |
| **Teacher**  | مَا العَدَدُ الَّذِي يَأْتِي بَعْدَ سِتَّةٍ؟                                                             | What number comes after six?                               |
| **Student**  | يَأْتِي بَعْدَهُ سَبْعَةٌ.                                                                               | Seven comes after it.                                      |
| **Teacher**  | صَحِيحٌ!                                                                                                 | Correct!                                                   |
| **Teacher**  | مَا العَدَدُ الَّذِي قَبْلَ خَمْسَةٍ؟                                                                    | What number comes before five?                             |
| **Student**  | أَرْبَعَةٌ.                                                                                              | Four.                                                      |
| **Teacher**  | أَحْسَنْتَ!                                                                                              | Well done!                                                 |
| **Teacher**  | كَمْ يَكُونُ اِثْنَانٌ فِي كُلِّ يَدٍ؟                                                                   | How many are in each hand?                                 |
| **Student**  | اِثْنَانِ.                                                                                               | Two.                                                       |
| **Teacher**  | إِذًا كَمْ يَكُونُ المَجْمُوعُ؟                                                                          | So, what is the total?                                     |
| **Student**  | أَرْبَعَةٌ.                                                                                              | Four.                                                      |
| **Teacher**  | أَحْسَنْتَ، أَرْبَعَةٌ أَصَابِعَ!                                                                        | Great, four fingers!                                       |
| **Teacher**  | كَيْفَ نَكْتُبُ الرَّقْمَ ثَمَانِيَةً؟                                                                   | How do we write the number eight?                          |
| **Student**  | نَرْسُمُ دَائِرَتَيْنِ مَرْبُوطَتَيْنِ.                                                                  | We draw two connected circles.                             |
| **Teacher**  | صَحِيحٌ! هَذَا هُوَ الرَّقْمُ ثَمَانِيَةٌ.                                                               | Correct! That’s the number eight.                          |
| **Teacher**  | مَنْ يَسْتَطِيعُ أَنْ يَكْتُبَ مُعَادَلَةً صَغِيرَةً؟                                                    | Who can write a small equation?                            |
| **Student**  | ثَلَاثَةٌ زَائِدٌ ثَلَاثَةٌ يُسَاوِي سِتَّةً.                                                            | Three plus three equals six.                               |
| **Teacher**  | مُمْتَازٌ!                                                                                               | Excellent!                                                 |
| **Teacher**  | كَيْفَ نَجْعَلُ الرَّقْمَ أَكْبَرَ؟                                                                      | How can we make the number bigger?                         |
| **Student**  | نُضِيفُ إِلَيْهِ رَقْمًا آخَرَ.                                                                          | We add another number to it.                               |
| **Teacher**  | وَكَيْفَ نَجْعَلُهُ أَصْغَرَ؟                                                                            | And how can we make it smaller?                            |
| **Student**  | نَطْرَحُ مِنْهُ.                                                                                         | We subtract from it.                                       |
| **Teacher**  | رَائِعٌ! أَنْتُمْ أَذْكِيَاءُ اليَوْمَ!                                                                  | Wonderful! You are smart today!                            |
| **Teacher**  | مَنْ يُرِيدُ أَنْ يَحْلَّ مَسْأَلَةً؟                                                                    | Who wants to solve a problem?                              |
| **Student**  | أَنَا! أَنَا أُرِيدُ!                                                                                    | Me! I want to!                                             |
| **Teacher**  | حَسَنًا، كَمْ يُسَاوِي سَبْعَةٌ نَاقِصٌ ثَلَاثَةٍ؟                                                       | Okay, what is seven minus three?                           |
| **Student**  | يُسَاوِي أَرْبَعَةً.                                                                                     | It equals four.                                            |
| **Teacher**  | إِجَابَةٌ صَحِيحَةٌ!                                                                                     | Correct answer!                                            |
| **Teacher**  | لِنَكْتُبِ الأَعْدَادَ عَلَى السَّبُّورَةِ.                                                              | Let’s write the numbers on the board.                      |
| **Students** | حَاضِرٌ يَا أُسْتَاذُ.                                                                                   | Yes, teacher.                                              |
| **Teacher**  | مَا هُوَ نَاتِجُ جَمْعِ اِثْنَيْنِ وَخَمْسَةٍ؟                                                           | What is the result of adding two and five?                 |
| **Student**  | اِثْنَانِ وَخَمْسَةٌ يُسَاوِي سَبْعَةً.                                                                  | Two and five equal seven.                                  |
| **Teacher**  | أَحْسَنْتِ يَا لَيْلَى!                                                                                  | Well done, Layla!                                          |
| **Teacher**  | لِنَتَأَكَّدْ مِنَ الإِجَابَةِ.                                                                          | Let’s check the answer.                                    |
| **Students** | نَعَمْ، هِيَ سَبْعَةٌ.                                                                                   | Yes, it’s seven.                                           |
| **Teacher**  | مُمْتَازٌ، كُلُّكُمْ فَهِمْتُمُ اليَوْمَ!                                                                | Excellent, you all understood today!                       |
