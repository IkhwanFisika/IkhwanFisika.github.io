### MSA Arabic Classroom Phrases Junior Secondary Level
| English Meaning | Arabic with Harakat | Example Sentence (Arabic with Harakat) | Example Sentence (English Translation) |
|-----------------|---------------------|---------------------------------------|---------------------------------------|
| Fraction | كَسْرٌ | اِكْتُبِ الكَسْرَ ٣/٤. | Write the fraction 3/4. |
| Numerator | البَسْطُ | البَسْطُ فِي الكَسْرِ ٥/٨ هُوَ ٥. | The numerator in the fraction 5/8 is 5. |
| Denominator | المَقَامُ | المَقَامُ فِي ٢/٧ هُوَ ٧. | The denominator in 2/7 is 7. |
| Proper fraction | كَسْرٌ حَقِيقِيٌّ | ٣/٥ كَسْرٌ حَقِيقِيٌّ. | 3/5 is a proper fraction. |
| Improper fraction | كَسْرٌ غَيْرُ حَقِيقِيٍّ | ٨/٣ كَسْرٌ غَيْرُ حَقِيقِيٍّ. | 8/3 is an improper fraction. |
| Mixed number | عَدَدٌ كَسْرِيٌّ مُخْتَلِطٌ | حَوِّلْ ٧/٤ إِلَى عَدَدٍ كَسْرِيٍّ مُخْتَلِطٍ. | Convert 7/4 to a mixed number. |
| Simplify the fraction | بَسِّطِ الكَسْرَ | بَسِّطِ الكَسْرَ ٦/٨. | Simplify the fraction 6/8. |
| Equivalent fractions | كُسُورٌ مُتَكَافِئَةٌ | ١/٢ وَ ٢/٤ كُسُورٌ مُتَكَافِئَةٌ. | 1/2 and 2/4 are equivalent fractions. |
| Lowest terms | أَبْسَطُ صُورَةٍ | اِكْتُبِ ٩/١٢ بِأَبْسَطِ صُورَةٍ. | Write 9/12 in lowest terms. |
| Common denominator | مَقَامٌ مُشْتَرَكٌ | اِجِدْ مَقَامًا مُشْتَرَكًا لِـ ١/٣ وَ ١/٦. | Find a common denominator for 1/3 and 1/6. |
| Least common multiple (LCM) | أَصْغَرُ مُضَاعَفٍ مُشْتَرَكٍ | أَصْغَرُ مُضَاعَفٍ مُشْتَرَكٍ لِـ ٤ وَ ٦ هُوَ ١٢. | The LCM of 4 and 6 is 12. |
| Add fractions | اِجْمَعِ الكُسُورَ | اِجْمَعِ الكُسُورَ: ١/٤ + ١/٤. | Add the fractions: 1/4 + 1/4. |
| Subtract fractions | اِطْرَحِ الكُسُورَ | اِطْرَحِ: ٥/٦ - ١/٦. | Subtract: 5/6 - 1/6. |
| Multiply fractions | اِضْرِبِ الكُسُورَ | اِضْرِبِ: ٢/٣ × ٣/٤. | Multiply: 2/3 × 3/4. |
| Divide fractions | اِقْسِمِ الكُسُورَ | اِقْسِمِ: ١/٢ ÷ ١/٤. | Divide: 1/2 ÷ 1/4. |
| Reciprocal | المَقْلُوبُ | المَقْلُوبُ لِـ ٣/٥ هُوَ ٥/٣. | The reciprocal of 3/5 is 5/3. |
| Decimal | عَشَرِيٌّ | ٠.٧٥ عَدَدٌ عَشَرِيٌّ. | 0.75 is a decimal. |
| Decimal point | الفَاصِلَةُ العَشْرِيَّةُ | ضَعِ الفَاصِلَةَ العَشْرِيَّةَ بَعْدَ رَقْمَيْنِ. | Place the decimal point after two digits. |
| Percentage | نِسْبَةٌ مِئَوِيَّةٌ | ٥٠% نِسْبَةٌ مِئَوِيَّةٌ. | 50% is a percentage. |
| Percent | بِالمِئَةِ | ٢٥ بِالمِئَةِ مِنْ ١٠٠ هُوَ ٢٥. | 25 percent of 100 is 25. |
| Convert to percent | حَوِّلْ إِلَى نِسْبَةٍ مِئَوِيَّةٍ | حَوِّلْ ٠.٧٥ إِلَى نِسْبَةٍ مِئَوِيَّةٍ. | Convert 0.75 to a percent. |
| Ratio | نِسْبَةٌ | النِّسْبَةُ بَيْنَ ٢ وَ ٤ هِيَ ١:٢. | The ratio between 2 and 4 is 1:2. |
| Proportion | تَنَاسُبٌ | ٢/٤ = ١/٢ تَنَاسُبٌ. | 2/4 = 1/2 is a proportion. |
| Solve the proportion | حَلِّ التَّنَاسُبَ | حَلِّ التَّنَاسُبَ: ٣/٥ = س/١٠. | Solve the proportion: 3/5 = x/10. |
| Integer | عَدَدٌ صَحِيحٌ | -٣ عَدَدٌ صَحِيحٌ. | -3 is an integer. |
| Positive integer | عَدَدٌ صَحِيحٌ مُوجَبٌ | ٥ عَدَدٌ صَحِيحٌ مُوجَبٌ. | 5 is a positive integer. |
| Negative integer | عَدَدٌ صَحِيحٌ سَالِبٌ | -٧ عَدَدٌ صَحِيحٌ سَالِبٌ. | -7 is a negative integer. |
| Absolute value | القِيمَةُ المُطْلَقَةُ | \| -٤ \| = ٤. | The absolute value of -4 is 4. |
| Number line | خَطُّ الأَعْدَادِ | ارْسُمْ -٢ عَلَى خَطِّ الأَعْدَادِ. | Plot -2 on the number line. |
| Coordinate plane | مُسْتَوَى الإِحْدَاثِيَّاتِ | ضَعِ النُّقْطَةَ (٣,٤) فِي مُسْتَوَى الإِحْدَاثِيَّاتِ. | Place the point (3,4) on the coordinate plane. |
| X-axis | مِحْوَرُ السِّينَاتِ | مِحْوَرُ السِّينَاتِ أُفُقِيٌّ. | The x-axis is horizontal. |
| Y-axis | مِحْوَرُ الصَّادَاتِ | مِحْوَرُ الصَّادَاتِ عَمُودِيٌّ. | The y-axis is vertical. |
| Origin | المَنْشَأُ | المَنْشَأُ هُوَ (٠,٠). | The origin is (0,0). |
| Quadrant | رُبْعٌ | النُّقْطَةُ (٢,٣) فِي الرُّبْعِ الأَوَّلِ. | The point (2,3) is in the first quadrant. |
| Ordered pair | زَوْجٌ مُرَتَّبٌ | (٥,-٢) زَوْجٌ مُرَتَّبٌ. | (5,-2) is an ordered pair. |
| Variable | مُتَغَيِّرٌ | س هُوَ مُتَغَيِّرٌ فِي المُعَادَلَةِ. | x is a variable in the equation. |
| Expression | تَعْبِيرٌ | ٣س + ٥ تَعْبِيرٌ. | 3x + 5 is an expression. |
| Equation | مُعَادَلَةٌ | ٢س + ٣ = ٧ مُعَادَلَةٌ. | 2x + 3 = 7 is an equation. |
| Solve the equation | حَلِّ المُعَادَلَةَ | حَلِّ المُعَادَلَةَ: س + ٤ = ٩. | Solve the equation: x + 4 = 9. |
| Like terms | حُدُودٌ مُتَشَابِهَةٌ | ٢س وَ ٥س حُدُودٌ مُتَشَابِهَةٌ. | 2x and 5x are like terms. |
| Combine like terms | اِجْمَعِ الحُدُودَ المُتَشَابِهَةَ | اِجْمَعِ: ٣س + ٢س - س. | Combine: 3x + 2x - x. |
| Coefficient | مُعَامِلٌ | مُعَامِلُ س فِي ٤س هُوَ ٤. | The coefficient of x in 4x is 4. |
| Constant | ثَابِتٌ | الثَّابِتُ فِي ٣س + ٥ هُوَ ٥. | The constant in 3x + 5 is 5. |
| Distributive property | الخَاصِّيَّةُ التَّوْزِيعِيَّةُ | ٢(س + ٣) = ٢س + ٦ بِالخَاصِّيَّةِ التَّوْزِيعِيَّةِ. | 2(x + 3) = 2x + 6 using the distributive property. |
| Exponent | أُسٌّ | الأُسُّ فِي ٥² هُوَ ٢. | The exponent in 5² is 2. |
| Base | الأَسَاسُ | الأَسَاسُ فِي ٤³ هُوَ ٤. | The base in 4³ is 4. |
| Power | قُوَّةٌ | ٣⁴ قُوَّةٌ. | 3⁴ is a power. |
| Square | تَرْبِيعٌ | تَرْبِيعُ ٦ هُوَ ٣٦. | The square of 6 is 36. |
| Cube | تَكْعِيبٌ | تَكْعِيبُ ٢ هُوَ ٨. | The cube of 2 is 8. |
| Square root | الجِذْرُ التَّرْبِيعِيُّ | الجِذْرُ التَّرْبِيعِيُّ لِـ ١٦ هُوَ ٤. | The square root of 16 is 4. |
| Order of operations | تَرْتِيبُ العَمَلِيَّاتِ | احْسُبْ بِتَرْتِيبِ العَمَلِيَّاتِ: ٢ + ٣ × ٤. | Calculate using order of operations: 2 + 3 × 4. |
| PEMDAS | (اِبْدَأْ بِالقَوْسَيْنِ، ثُمَّ الأُسُسِ، ثُمَّ الضَّرْبِ وَالقِسْمَةِ، ثُمَّ الجَمْعِ وَالطَّرْحِ) | تَذَكَّرْ PEMDAS لِحَلِّ المَسَائِلِ. | Remember PEMDAS to solve problems. |
| Evaluate | قَيِّمْ | قَيِّمْ ٢س + ٣ عِنْدَمَا س = ٤. | Evaluate 2x + 3 when x = 4. |
| Geometry | الهَنْدَسَةُ | نَدْرُسُ الهَنْدَسَةَ اليَوْمَ. | We are studying geometry today. |
| Point | نُقْطَةٌ | ضَعْ نُقْطَةً عَلَى الخَطِّ. | Put a point on the line. |
| Line | خَطٌّ | ارْسُمْ خَطًّا مُسْتَقِيمًا. | Draw a straight line. |
| Line segment | قِطْعَةٌ مُسْتَقِيمَةٌ | قِسْ قِطْعَةً مُسْتَقِيمَةً بِطُولِ ٥ سَم. | Measure a line segment of 5 cm. |
| Ray | شُعَاعٌ | الشُّعَاعُ يَبْدَأُ مِنْ نُقْطَةٍ وَيَمْتَدُّ. | A ray starts at a point and extends. |
| Angle | زَاوِيَةٌ | قِسْ الزَّاوِيَةَ بِالمِنْقَلَةِ. | Measure the angle with a protractor. |
| Degree | دَرَجَةٌ | الزَّاوِيَةُ ٩٠ دَرَجَةٌ. | The angle is 90 degrees. |
| Acute angle | زَاوِيَةٌ حَادَّةٌ | زَاوِيَةٌ أَصْغَرُ مِنْ ٩٠ دَرَجَةٍ حَادَّةٌ. | An angle less than 90 degrees is acute. |
| Right angle | زَاوِيَةٌ قَائِمَةٌ | الزَّاوِيَةُ القَائِمَةُ ٩٠ دَرَجَةٌ. | A right angle is 90 degrees. |
| Obtuse angle | زَاوِيَةٌ مُنْفَرِجَةٌ | زَاوِيَةٌ أَكْبَرُ مِنْ ٩٠ دَرَجَةٍ مُنْفَرِجَةٌ. | An angle greater than 90 degrees is obtuse. |
| Straight angle | زَاوِيَةٌ مُسْتَقِيمَةٌ | الزَّاوِيَةُ المُسْتَقِيمَةُ ١٨٠ دَرَجَةٌ. | A straight angle is 180 degrees. |
| Vertex | رَأْسٌ | رَأْسُ الزَّاوِيَةِ هُنَا. | The vertex of the angle is here. |
| Parallel lines | خُطُوطٌ مُتَوَازِيَةٌ | هَذَانِ الخَطَّانِ مُتَوَازِيَانِ. | These two lines are parallel. |
| Perpendicular lines | خُطُوطٌ مُتَعَامِدَةٌ | الخَطَّانِ مُتَعَامِدَانِ عِنْدَ ٩٠ دَرَجَةٍ. | The lines are perpendicular at 90 degrees. |
| Triangle | مُثَلَّثٌ | ارْسُمْ مُثَلَّثًا مُتَسَاوِيَ الأَضْلَاعِ. | Draw an equilateral triangle. |
| Scalene | مُتَفَاوِتُ الأَضْلَاعِ | المُثَلَّثُ مُتَفَاوِتُ الأَضْلَاعِ. | The triangle is scalene. |
| Isosceles | مُتَسَاوِيُ السَّاقَيْنِ | المُثَلَّثُ مُتَسَاوِيُ السَّاقَيْنِ. | The triangle is isosceles. |
| Equilateral | مُتَسَاوِيُ الأَضْلَاعِ | كُلُّ أَضْلَاعِهِ مُتَسَاوِيَةٌ. | All sides are equal. |
| Sum of angles in triangle | مَجْمُوعُ زَوَايَا المُثَلَّثِ | مَجْمُوعُ زَوَايَا المُثَلَّثِ ١٨٠ دَرَجَةٌ. | The sum of angles in a triangle is 180 degrees. |
| Perimeter | المُحِيطُ | احْسُبْ مُحِيطَ المُسْتَطِيلِ. | Calculate the perimeter of the rectangle. |
| Area | المَسَاحَةُ | مَسَاحَةُ المُرَبَّعِ = ضَلْعٌ × ضَلْعٌ. | The area of a square = side × side. |
| Area of rectangle | مَسَاحَةُ المُسْتَطِيلِ | مَسَاحَةُ المُسْتَطِيلِ = طُولٌ × عَرْضٌ. | Area of rectangle = length × width. |
| Area of triangle | مَسَاحَةُ المُثَلَّثِ | مَسَاحَةُ المُثَلَّثِ = ١/٢ × قَاعِدَةٌ × ارْتِفَاعٌ. | Area of triangle = ½ × base × height. |
| Base | القَاعِدَةُ | القَاعِدَةُ ٦ سَم. | The base is 6 cm. |
| Height | الارْتِفَاعُ | الارْتِفَاعُ ٤ سَم. | The height is 4 cm. |
| Volume | الحَجْمُ | احْسُبْ حَجْمَ المُكَعَّبِ. | Calculate the volume of the cube. |

| Rectangular prism | مُتَوَازِيُ المُسْتَطِيلَاتِ | هَذَا مُتَوَازِيُ المُسْتَطِيلَاتِ. | This is a rectangular prism. |
| Cylinder | أُسْطُوانَةٌ | الأُسْطُوانَةُ لَهَا قَاعِدَتَانِ دَائِرِيَّتَانِ. | A cylinder has two circular bases. |
| Data | بَيَانَاتٌ | اجْمَعِ البَيَانَاتِ مِنَ الصَّفِّ. | Collect data from the class. |
| Graph | رَسْمٌ بَيَانِيٌّ | ارْسُمْ رَسْمًا بَيَانِيًّا عَمُودِيًّا. | Draw a bar graph. |
| Bar graph | رَسْمٌ بَيَانِيٌّ عَمُودِيٌّ | الرَّسْمُ البَيَانِيُّ العَمُودِيُّ يُظْهِرُ الأَعْدَادَ. | The bar graph shows the numbers. |
| Line graph | رَسْمٌ بَيَانِيٌّ خَطِّيٌّ | الرَّسْمُ الخَطِّيُّ يُظْهِرُ التَّغَيُّرَ. | The line graph shows change. |
| Pie chart | رَسْمٌ بَيَانِيٌّ دَائِرِيٌّ | الرَّسْمُ الدَّائِرِيُّ يُظْهِرُ النِّسَبَ. | The pie chart shows percentages. |
| Mean | المُتَوَسِّطُ الحِسَابِيُّ | المُتَوَسِّطُ = المَجْمُوعُ ÷ العَدَدُ. | Mean = sum ÷ count. |
| Median | المُتَوَسِّطُ | رَتِّبِ الأَعْدَادَ ثُمَّ اِجِدِ المُتَوَسِّطَ. | Arrange the numbers then find the median. |
| Mode | الوَضْعُ | الوَضْعُ هُوَ العَدَدُ الأَكْثَرُ تَكْرَارًا. | The mode is the most frequent number. |
| Probability | الاحْتِمَالُ | احْتِمَالُ رَمْيِ عُمْلَةٍ وَظُهُورِ كِتَابَةٍ ١/٢. | The probability of heads on a coin is 1/2. |
| Certain | مُؤَكَّدٌ | ظُهُورُ الشَّمْسِ صَبَاحًا مُؤَكَّدٌ. | The sun rising is certain. |
| Impossible | مُسْتَحِيلٌ | رَمْيُ نَرْدٍ وَظُهُورُ ٧ مُسْتَحِيلٌ. | Rolling a 7 on a die is impossible. |
| Likely | مُحْتَمَلٌ | هُطُولُ المَطَرِ اليَوْمَ مُحْتَمَلٌ. | Rain today is likely. |

| Experiment | تَجْرِبَةٌ | نُجْرِي تَجْرِبَةً بِرَمْيِ النَّرْدِ. | We conduct an experiment by rolling a die. |
